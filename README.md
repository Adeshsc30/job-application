#Job-Tracker
  =>A full-stack web application to track job applications through different hiring stages.

#Screenshots
  => Made a folder (Can watch from there)

#Tech Stack
| Layer    | Technology                                   |
| -------- | -------------------------------------------- |
| Frontend | React 18, TypeScript, Tailwind CSS v4, Vite  |
| Backend  | Node.js, Express, TypeScript                 |
| Database | PostgreSQL (Neon.tech cloud)                 |
| ORM      | Prisma                                       |
| API      | REST                                         |

#Features
- View all job applications in a clean table
- View full details of any application
- Add new applications with a modal form
- Edit existing applications
- Delete applications with a confirmation step
- Filter by status — Applied, Interviewing, Offer, Rejected
- Search by company name or job title
- Loading states and error messages
- Fully typed with TypeScript
- Responsive design
